# Tar.gz Files

### List contents
```bash
tar -tvf archive.tar.gz
```

### Compress Tar.gz
```bash
#   -args output  filename/directory to tar
tar -cvzf archive filename
```

### Extract tar.gz

```bash
# -C flag to specify output 
tar -xvf archive.tar.gz -C /home/linuxize/files
```
